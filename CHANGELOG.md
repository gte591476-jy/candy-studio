# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-27

### Added

- Public GitHub repository initialization
- MySQL-backed website runtime
- User registration and login flow
- Admin dashboard
- Points, recharge, and membership system
- Banana-style image and video generation workflow
- Bilingual documentation

### Changed

- Removed low-price GeminiGen-specific module and related backend/frontend entry points
- Standardized project positioning around Banana Pro / Banana 2 resale website use cases
- Reworked authentication storage and cache behavior to reduce account mix-up risk

### Security

- Replaced committed local secrets with placeholders
- Improved cache-control behavior for authenticated API responses
